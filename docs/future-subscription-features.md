# Future Subscription Features & Tasks

## Plans & Features

### Free Plan
- Up to 10 users per building/organization
- Website dashboard access
- Core scheduling features

### Basic Plan ($3/user/month)
- Website dashboard access
- Email notifications
- Core scheduling features

### Pro Plan ($6/user/month)
- Website dashboard access
- Email notifications
- Mobile notifications / app access
- Shift trading
- Audit logs
- Priority support

## Implementation Tasks
1. Track number of users per building (organization)
2. Enforce free plan limit (10 users/building)
3. Implement paid plans (Basic, Pro) and feature restrictions
4. Integrate payment processing (Stripe, PayPal, etc.)
5. Add subscription status and plan management (upgrade/downgrade/cancel)
6. Restrict features/UI based on plan (e.g., app access, notifications, trading, audit logs)
7. Admin dashboard for billing, usage, and plan management
8. Automated email reminders for billing and plan changes
9. API endpoints for subscription management
10. UI for users/admins to view and manage their plan
11. Implement priority support system for Pro users

---
This list can be expanded as new requirements arise. Each item can be tracked and checked off as implemented.
